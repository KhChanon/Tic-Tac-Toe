# Tic-Tac-Toe
#### Made for CPE342 Java Programming Language class Assignment 2
#### A Tic Tac Toe game
## Features
<ul>
  <li>Have Player vs Player</li>
  <li>Have Player vs Bot</li>
</ul>

## Running
Run the command in terminal in the following order

```
$   javac game.java
$   java MyApp.java
```
